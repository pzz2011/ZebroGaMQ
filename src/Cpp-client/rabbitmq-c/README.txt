
This is a C-language AMQP client library for use with v2.0+ of the RabbitMQ broker. Its website: https://github.com/alanxz/rabbitmq-c

HOWTOS:
+ Download code from https://github.com/alanxz/rabbitmq-c
+ Copy these files from folders librabbitmq & librabbitmq/unix

    amqp.h
    amqp_ssl_socket.h
    amqp_timer.h
    threads.h
    amqp_connection.c
    amqp_mem.c
    amqp_table.c
    amqp_url.c
    amqp_consumer.c
    amqp_private.h
    amqp_tcp_socket.c
    amqp_framing.c
    amqp_socket.c
    amqp_tcp_socket.h
    amqp_framing.h
    amqp_socket.h
