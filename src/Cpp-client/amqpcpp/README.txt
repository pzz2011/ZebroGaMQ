
amqpcpp is a C++ library for Message Queue Server RabbitMQ. It's the C++ wrapper of rabbitmq-c (https://github.com/alanxz/rabbitmq-c). Its website: https://github.com/akalend/amqpcpp

HOWTOS:
+ Download code from https://github.com/akalend/amqpcpp
+ Get code files from /src and the header file from /include
