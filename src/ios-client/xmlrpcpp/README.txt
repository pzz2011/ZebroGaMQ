
XMLRPCPP is a C++ implementation of the XML-RPC protocol taken from the site: http://xmlrpcpp.sourceforge.net/

HOWTOS:
+ Download code from http://sourceforge.net/projects/xmlrpcpp/files/xmlrpc%2B%2B/Version%200.7/xmlrpc%2B%2B0.7.zip/download
+ Copy all code files from its /src folder and then put in /xmlrpcpp folder
